BlockchainTokenSniper V3 ships with multiple EVM networks plus Solana ready to use. Each chain can be selected from **Change Chain / DEX** inside the Telegram bot, and advanced users can override RPC nodes or router addresses if needed.

<Note>
  The lists below reflect the default `network_config.json` bundled with V3. Always double-check the bot interface for the latest additions or experimental networks.
</Note>

## Solana network

| Component | Details |
| --- | --- |
| Execution venue | Jupiter (Jup.ag) |
| How it's used | All swaps, price lookups, and route optimizations flow through Jupiter to guarantee best execution across Solana DEXs. |

<Note>
  Additional EVM chains and DEXs can be enabled per user request or future releases. When in doubt, look at the Telegram interface—its dropdown is always the source of truth.
